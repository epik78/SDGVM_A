Initialise cohorts for the gridcell prior to year loop.

! Initializes number of cohors and system state for each ft that is present in year 1,checks cluse(ft,1)>0.0. For each ft create as many cohorts as its lifespan in years given by
pftmort.Initial cover for each cohort is given by the ft total
cover of year 1 divided by the number of ft cohorts.It has previously read the pft parameters for each ft from file and assigned them to variable pft_tab(ft).Here is assigns the correct parameters to the same variable type pft(cohort) for each cohort.

subroutine read_input::read_param ( character(len=str_len)  stver)

Read internal parameters from "param.dat" file, and io parameters from "misc_params.dat".

First reads misc_params.dat file in /f03 with input on year step to output on screen and whether run is regional or country.Saved in msp structure defined in misc_parameters.f90. It then reads the param.dat file in /inc with the tuning parameters and saves in structure tgp defined in tuning_parameters.f90.
